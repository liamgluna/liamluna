# Liam Luna

Personal portfolio site for Liam Luna.

## Technologies used

- React
- TypeScript
- Vite
- React Router
- Tailwind CSS

## Setup / development

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Live website

- https://liamluna.com
- Deployed via Cloudflare Workers
